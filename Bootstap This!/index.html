Intro

What is Bootstrap

Bootstrap is a coding framework used for quickly developing responsive mobile layouts. The founders of Bootstrap, Mark Otto & Jacob Thornton, describe their product as "The most popular HTML, CSS, and JS framwork for developing responsive, mobile first projects on web." But, what does that all mean?

A coding framework is defined by Brooke Campbell as: "A framework is a collection of programs brought together to accomplish a task, make coding more efficient and generally make the lives of coders easier." Therfore, we can construe that Boostrap is a collection of code made to make coding a website easier for the coder.

Instaling and using Bootstrap

Here is a step by step tutorial for how to initialize bootstrap in to your code.

  1. Head over the the Bootstrap website.
  2. Click on the Download button. This will bring you to the Boostrap dowlad page where you will be able to dowload all nesesary files from Bootstrap.
  3. Once on the downlad page you will have 3 main options on how to apply boostrap to your project; Compiled CSS and JS, Source Files, or Bootstrap CDN.

    1. The simplest option is to coppy/paste the Bootstrap CDN code lines at the top of your page
      • This will directly link your project to the Bootstrap library letting you utelise it's library without downlad.

    2. Second option is to dowload the Compiled CSS and JS. then find the downladed folder, open it and link the CSS & JS files from that folder to your project similarly done in the CDN code.

    3. Third option is to download the Source Files. This option will give you a folder with all of the Bootstrap SASS files. You will need to link the CSS once again to your project at the top with the <link rel:"stylesheet" href:...> just like both other options.
      • This permits for greater control over the source files, however should not be used by novice coders who do not understand SASS.

  4. Once the files downloaded & linked to your html document, you may start coding your base HTML.
  5. Bootstrap usses a series of classes to identify and apply properties to your code. In order to utilize Bootstrap fully you should familiorize yourself with classes used by Bootstrap.
  6. Utilize the "Classes" portion below to help familiorize yourself with basic tools.
